# 文档标题：SMT贴片机报警手册 - E302 真空压力不足
文档编号：AS-MFG-KB-ALARM-SMTX100-E302
文档版本：v1.0
文档类型：报警手册
设备型号：SMT-X100
适用产线：电子装配线
故障代码：E302
标签：真空不足, 吸嘴异常, 抛料, 贴片失败

## 故障定义
E302表示贴片机吸嘴真空压力低于安全运行阈值，设备无法稳定吸附元件，可能导致抛料、贴装偏移或停机。

## 典型现象
- 设备报警代码显示E302
- 吸嘴吸取元件失败
- 抛料率上升
- 连续贴片失败
- 真空压力读数偏低

## 可能原因
1. 吸嘴堵塞或磨损
2. 真空过滤器污染
3. 气路接头松动导致漏气
4. 真空电磁阀动作异常
5. 真空泵性能下降

## 推荐排查顺序
1. 检查吸嘴表面是否堵塞、磨损或变形
2. 检查真空压力表读数
3. 检查过滤器是否积尘
4. 检查气路接头及软管是否漏气
5. 检查真空电磁阀响应
6. 检查真空泵输出

## 处理建议
- 清洁或更换吸嘴
- 更换真空过滤芯
- 紧固气路接头
- 更换故障电磁阀
- 对真空泵进行维护或更换

## 安全提醒
排查前必须停机并完成泄压操作。禁止在带压状态下拆卸气路组件。

## 相关备件
- 吸嘴：NZ-14A
- 过滤芯：VF-02
- 电磁阀：SV-11

## 相似关键词
E302, 真空不足, 吸嘴堵塞, 抛料, 吸取失败
