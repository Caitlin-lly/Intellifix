# 文档标题：故障知识卡-E302真空压力不足
文档编号：AS-MFG-KB-CARD-SMTX100-E302-QUICK
文档版本：v1.0
文档类型：知识卡片
标签：知识卡, E302, 快速排查

## 故障名称
E302 真空压力不足

## 高概率原因
1. 吸嘴堵塞
2. 吸嘴磨损
3. 过滤器堵塞
4. 气路漏气
5. 电磁阀异常

## 推荐排查顺序
吸嘴 → 过滤器 → 气路 → 电磁阀 → 真空泵

## 一线处理建议
- 先清洁吸嘴
- 无效则更换吸嘴
- 继续无效则检查过滤器污染
- 若仍未恢复，联系高级工程师

## 升级条件
- 连续处理15分钟未恢复
- 同一故障一周内重复3次
- 真空泵疑似异常
- 存在安全风险

## 风险级别
中

## 处理目标
在10分钟内完成初步诊断，在20分钟内恢复生产或完成升级。
